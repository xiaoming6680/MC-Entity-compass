# 生物指南针使用说明

## 运行要求
- Minecraft `1.21.11`
- Fabric Loader `0.18.4` 或更高版本
- Fabric API
- Java 21

多人游戏需要服务端安装 MOD，想使用菜单和 HUD 的客户端也需要安装 MOD。

## 安装方法
1. 将 `entity-compass-1.0.6.jar` 放入 `mods` 文件夹。
2. 确认 Fabric API 也在 `mods` 文件夹中。
3. 使用 Fabric 启动游戏。

## 合成配方

配方：

```text
 E
ECE
 E
```

- `C`：指南针
- `E`：末影珍珠

## 选择目标
- 手持生物指南针。
- 右键指南针打开目标菜单。
- 可以选择一种生物，也可以选择在线玩家。
- 选择生物时，指南针会锁定范围内最近的同类已加载生物。
- 选择玩家时，指南针会锁定指定在线玩家。

## 全局设置
按 `M` 打开全局设置菜单。

这些设置由服务端统一控制，会影响所有玩家的生物指南针：
- 附魔光效：开启或关闭外观上的附魔光效。
- 距离显示：关闭、物品栏上方、准星上方、准星下方。
- 近距离失效：开启后，如果最近目标距离小于设置格数，指南针会停止指向目标。

多人服务器中，只有管理员可以修改这些全局设置。

近距离阈值可填写 `0` 到 `255`。

## HUD 提示
- 距离文字会显示当前目标和距离。
- `目标在附近！` 表示最近目标小于设置阈值。
- `找不到目标！` 表示当前选择的目标不可用，或不在追踪范围内。

指南针永远先判断最近的匹配目标。如果最近目标小于阈值，会显示近距离提示，不会跳过它去锁定更远的目标。
